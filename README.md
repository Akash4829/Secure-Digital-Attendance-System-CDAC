# Secure Digital Attendance System

## Tech Stack
- Backend: Spring Boot (Maven)
- Frontend: React (Vite)
- Database: MySQL
- Authentication: JWT

## Features
- User authentication (login/register)
- Attendance management
- OTP-based password reset
- Role-based access

## Project Structure
Backend/   → Spring Boot REST API  
Frontend/  → React Vite UI  

## Run Backend
cd Backend
mvn spring-boot:run

## Run Frontend
cd Frontend
npm install
npm run dev
